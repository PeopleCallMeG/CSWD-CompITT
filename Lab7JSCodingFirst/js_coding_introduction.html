Tools for Javascript Development

The browser developer tools

  • Javascript console
    • To access the Javascript console in your browser, press key F12 to open the developer tools, then click the Console tab.
    • The Javascript interpreter inside the browser uses the Console to communicate (e.g. with information, warning and error messages).
    • You as the developer can use the console to interactively run Javascript code.
  • The debugger
    • To access the Javascript debugger in your browser, press key F12 to open the developer tools, then click the Debugger tab.
    • Use the Sources panel to navigate through the file system and open files.
    • Once a file is open, you can click on line numbers in the column to the left to set breakpoints.

Linting tools

A linting tool checks the style, syntax and semantics of code. You will find an online Javascript linter here.
