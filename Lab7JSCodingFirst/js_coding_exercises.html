Exercise 1

The following html elements are defined: a paragraph with a single letter, A, and three buttons: "Change to A", "Change to B" and "Change to C".

Your task is to add inline Javascript code to the three buttons so that each, when clicked, does what its name says it does i.e. the button "Change to A" changes the letter to 'A' etc. Test your solution.

Exercise 2

A button called "Disable/enable 'Change to C'" is defined.

Your task is to define a function that toggles the 'disabled' status of the 'Change to C' button (when 'Change to C' is enabled, as it is at the start, the function disables it and vice versa). Define the function internally in this HTML file (in the script element inside the file's head element). Then add an event listener inline for the "Disable/enable 'Change to C'" button so that clicking on it results in a call to the function you just defined. Test your solution.

Exercise 3

An element with id "magic-element" and containing text "Magic text" is defined.

Your task is to write a function that changes the content of this element to the image "logo.png", supplied for the exercise. Define the function internally in this HTML file. Then write code that programmatically adds the defined function as an event listener for the 'mouseover' event on the element. Get this code to be called once the entire HTML page is loaded. Test your solution.
